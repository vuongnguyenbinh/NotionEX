Advanced Setup

For power users who want more control.

üìÅ Use existing Notion databases

Instead of creating new databases, you can connect to existing ones:

  1. Go to Settings ‚Üí Database Mapping
  2. Click "Use existing database" for each type
  3. Select your database from the dropdown
  4. Map the properties (we'll guide you)
Requirements:
  • Tasks: Name, Checkbox, Select, Date properties
  • Bookmarks: Name, URL, Multi-select properties
  • Notes: Any database with Name property
  • Prompts: Any database with Name property
‚å®Ô∏è Customize keyboard shortcuts
  1. Go to chrome://extensions/shortcuts
  2. Find "Focus to Notion"
  3. Click the pencil icon to change any shortcut
üì• Import existing data

Have bookmarks in another tool? Import them:

  1. Export from your current tool (CSV format)
  2. In extension: Settings ‚Üí Import Data
  3. Upload CSV file
  4. Map columns to fields
  5. Click Import

Supported imports from: Pocket, Raindrop, Browser bookmarks, CSV

üîÑ Multiple Notion workspaces

Currently, you can only sync to one workspace at a time.

Workaround:

  • Use Notion's native sharing to sync databases between workspaces
  • Or use browser profiles with separate extension instances

Multi-workspace native support: Coming in v2.0

üíæ Backup your local data

Even with Notion sync, you might want local backups:

  1. Settings ‚Üí Data Management
  2. Click "Export All Data"
  3. Choose format: JSON or CSV
  4. Save file to your computer

Recommended: Set a monthly reminder to backup.

Troubleshooting Guide

Having issues? Find your solution below.

Connection Issues

Problem: "Failed to connect to Notion"

  • Check if popup was blocked (allow popups for extension)
  • Make sure you're logged into Notion in browser
  • Try logging out of Notion and back in
  • Clear extension data: Settings ‚Üí Reset ‚Üí Try again

Problem: "No workspaces found"

  • Make sure you have at least one Notion workspace
  • Check that you're logged into the correct Notion account
  • If using SSO, ensure browser session is active

Sync Issues

Problem: "Sync stuck on pending"

  • Check internet connection
  • Click manual sync button (üîÑ icon)
  • Check Notion API status: status.notion.so
  • Review Settings ‚Üí Sync Log for errors

Problem: "Data not appearing in Notion"

  • Wait 5 minutes (sync interval)
  • Force sync: Settings ‚Üí Sync Now
  • Check correct workspace is connected
  • Verify databases weren't accidentally deleted

Problem: "Duplicate entries in Notion"

Cause: Usually from interrupted sync

  • Delete duplicates manually in Notion
  • Force full sync: Settings ‚Üí Full Resync
  • Check for sync conflicts in log

Performance Issues

Problem: "Extension is slow"

  • Check how many items you have (1000+ may slow down)
  • Try archiving old completed tasks
  • Clear browser cache
  • Disable and re-enable extension

Problem: "Browser using too much memory"

  • Close extension sidebar when not using
  • Reduce sync frequency to manual
  • Archive old data to Notion only
