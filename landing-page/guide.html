Setup Guide

Get started with Focus to Notion in just a few minutes. Follow these simple steps.

1. Install the Extension

1

Visit Chrome Web Store

Click the button below to open Focus to Notion on Chrome Web Store.

Open Chrome Web Store
2

Add to Chrome

Click "Add to Chrome" button and accept the permissions. The extension needs these permissions to work:

  • sidePanel: Display the sidebar interface
  • storage: Save your data locally
  • alarms: Auto-sync with Notion
  • activeTab: Quick bookmark from current page
3

Pin to Toolbar

Click the puzzle icon in Chrome toolbar, then click the pin icon next to "Focus to Notion" for quick access.

4

Open Sidebar

Click the Focus to Notion icon to open the sidebar. You can now start adding tasks, bookmarks, and notes!

2. Connect to Notion (Optional)

Notion sync is optional. The extension works perfectly offline without it. If you want to sync with Notion, follow these steps:

1

Create Notion Integration

Go to Notion's integration page and create a new integration:

notion.so/my-integrations
  • Click "+ New integration"
  • Name it "Focus to Notion"
  • Select your workspace
  • Click "Submit"
  • Copy the "Internal Integration Token" (starts with secret_)
2

Create Database in Notion

Create a new database in Notion with these properties:

Property Name Type Description
Title Title Item title (default)
Type Select task, bookmark, note
Content Text Detailed content
URL URL Link (for bookmarks)
Priority Select high, medium, low
Deadline Date Due date
Completed Checkbox Task completion status
Tags Multi-select Labels
LocalID Text Local ID for sync
3

Share Database with Integration

Open your database in Notion, then:

  • Click "..." (top right corner)
  • Select "Connections"
  • Find and select "Focus to Notion"
  • Confirm sharing
4

Get Database ID

Open your database in browser and copy the Database ID from URL:

https://www.notion.so/workspace/abc123def456...?v=...

The Database ID is the 32-character string after your workspace name and before "?v="

5

Configure in Extension

Open the extension sidebar, go to Settings (gear icon), then:

  • Paste your Integration Token
  • Paste your Database ID
  • Click "Test" to verify connection
  • Click "Save" to save settings
6

Start Syncing!

The extension will automatically sync every 5 minutes. You can also click the sync icon in the header to sync manually.

Frequently Asked Questions

?

Will I lose data if I uninstall the extension?

Data is stored in Chrome's IndexedDB. If you clear browser data, local data will be lost. Connect to Notion for backup.

?

Why does Notion connection fail?

Check: (1) Token is correct and not expired, (2) Database ID format is correct, (3) Database is shared with your integration.

?

How do I know sync is working?

The sync icon in the header spins during sync. Check "Last synced" timestamp in Settings.

?

Can I use multiple Notion databases?

Currently supports one database for items (tasks/bookmarks/notes) and one for prompts. Multi-database support is planned.
